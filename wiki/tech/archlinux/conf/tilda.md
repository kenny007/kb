# tilda

## Install
```sh
kur tilda
```

## Config
```txt
general:
start tilda hidden
last terminal closed > new & hide

apperanece:
height: 50%
expand tabs
show single tabs
location: bottom

scrollbar:
limit scroll: none

keybindings:
open: f12
```
